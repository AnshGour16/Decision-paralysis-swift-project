# Decision Paralysis

Decision Paralysis is an app I built to help people know what to do when a disaster hits. Sometimes in an emergency, people freeze up because they don't know the right steps. This app gives clear info and interactive tools to help stay prepared.

## My Features

### 🌪️ The Disaster Guide
I've included guides for things like Earthquakes, Floods, and Fires. 
- **What to do:** Simple steps like "Drop, Cover, and Hold On".
- **Interactive Demos:** I added small interactive parts to each page so you can see how things like tsunamis or earthquakes work.

### 🎮 Simulator Game
I made a quiz-style simulator to test how you'd react in real situations. It’s meant to help you practice making quick decisions before an actual emergency happens.

### 📍 Family Rally Point
This is a tool to set a meeting place with your family. 
- **Map:** You can drop a pin on the map to save a spot.
- **Offline:** The location saves on your phone so you can check it even if the internet goes out during a storm.

### 🎒 Emergency Kit
A checklist for your "Go-Bag". It has all the basics like water and masks, but you can also add your own items.

## How I Built It
- **SwiftUI:** I used SwiftUI for the whole interface to keep it simple and clean.
- **MapKit:** This powers the Rally Point map.
- **UserDefaults:** I used this to save the kit and the rally point offline.
- **Design:** I went with a bold, high-contrast look so it's easy to read even if you're in a hurry or stressed.
